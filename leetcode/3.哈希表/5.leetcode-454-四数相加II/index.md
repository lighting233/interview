### 1.四个数组两两分组，每两个数组里的元素相加，分组之间匹配看是否得 0

### 2.为什么用 map 结构

- 假如一二数组里的两个元素相加为 a+b，那么我们既要存储这个数值的和，也要统计这个和出现的次数
- 后两个组合里查找 0 - （c+d），因为 0-（c+d）= a+b

### 复杂度n^2^  (n^2^ + n^2^)

 - 如果统计 a 和b,c,d之间的和，则时间复杂度为n^3^
